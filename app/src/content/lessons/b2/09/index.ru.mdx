## Деловое общение: встречи и email

Проводить деловые встречи: предлагать, уточнять, возражать конструктивно, подводить итоги; писать follow‑up письма с решениями и следующими шагами.

## Встреча: роли и повестка

- Rollen: voorzitter (ведущий), notulist (секретарь), deelnemers (участники)
- Agenda: opening → mededelingen → agendapunten → rondvraag → afsluiting
Фразы:
- Открытие: Zullen we beginnen? / Het doel van vandaag is...
- Предложение: Ik stel voor dat... / Laten we overwegen om...
- Уточнение: Bedoel je dat...? / Kun je dat toelichten?
- Несогласие: Ik ben het daar niet mee eens, omdat... / Een alternatief is...
- Итог: Samenvattend besluiten we dat... / De vervolgstappen zijn...

## Email: структура follow‑up

- Aanhef → Dank/Referentie → Besluiten → Actiepunten + eigenaar → Termijnen → Afsluiting
Шаблон:
Onderwerp: Follow‑up vergadering [datum]
Geachte/Beste ...,
Naar aanleiding van onze vergadering bevestig ik de besluiten:
1) ... 2) ... Actiepunten: A (eigenaar, termijn), B (...)
Met vriendelijke groet,
...

## Мини‑сценка

```
A (voorzitter): Het doel is prioriteiten bepalen.
B: Ik stel voor dat we eerst project X afronden.
C: Ik ben het daar deels mee eens; echter is de capaciteit beperkt.
A: Samenvattend: X afronden; bovendien starten we een kleine pilot voor Y.
```

## Упражнение 2: Преобразуй неформальное в формальное

| Неформально | Формально |
|-------------|-----------|
| Hoi allemaal, laten we starten. | -> `_______` |
| Ik vind dat we X moeten doen. | -> `_______` |
| Dat snap ik niet. | -> `_______` |
| Oké, we zijn klaar. | -> `_______` |
<details>
<summary>Ответы</summary>
1. Goedemiddag, zullen we beginnen?
2. Ik stel voor dat we X uitvoeren.
3. Kunt u dat nader toelichten?
4. Hiermee sluiten we de vergadering af.
</details>

## Упражнение 3: Schrijf een follow-up email

- Aanhef + referentie naar de vergadering
- 2 besluiten
- 2 actiepunten met eigenaar en termijn
- Afsluiting

## Словарь урока (20–25)

- vergadering — встреча
- voorzitter — ведущий
- notulist — секретарь
- agendapunt — пункт повестки
- mededeling — объявление
- rondvraag — вопросы в конце
- afsluiting — завершение
- voorstel doen — сделать предложение
- toelichten — пояснять
- bezwaar/bedenking — возражение/сомнение
- alternatief — альтернатива
- consensus — согласие
- besluit — решение
- actiepunt — пункт действия
- eigenaar (owner) — ответственный
- termijn — срок
- prioriteit — приоритет
- capaciteit — мощность/ресурс
- follow‑up — последующие действия
- bevestigen — подтверждать
- naar aanleiding van — в связи с
- onderwerp — тема письма
- bijgevoegd — приложенный
- samenvatten — подытоживать
- vastleggen — зафиксировать
