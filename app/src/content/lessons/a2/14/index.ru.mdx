## Грамматический чек-лист:

#### Времена глаголов:
- [ ] Могу рассказать что делал вчера (Perfectum)
- [ ] Могу описать свои планы (gaan + infinitief)
- [ ] Правильно выбираю hebben/zijn
- [ ] Знаю неправильные глаголы (zijn-geweest, gaan-gegaan)
#### Модальные глаголы:
- [ ] Различаю kunnen/mogen/moeten/willen
- [ ] Могу попросить вежливо (Mag ik...?)
- [ ] Выражаю необходимость (Ik moet...)
#### Сравнения:
- [ ] Сравниваю с dan (groter dan)
- [ ] Выражаю равенство с als (zo groot als)
- [ ] Знаю неправильные формы (goed-beter-beste)
#### Предлоги:
- [ ] Правильно использую in/op/aan для места
- [ ] Знаю временные предлоги (om/op/in)
- [ ] Описываю направления (naar/van/uit)
