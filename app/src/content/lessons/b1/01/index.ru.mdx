## Сложные предложения и союзы

**Научиться строить сложные предложения с различными союзами для выражения сложных идей и логических связей.**

## Союзы для сложных предложений

#### 1⃣ **Причина и следствие (Oorzaak en gevolg)**
##### **Omdat (потому что)**
```
Ik leer Nederlands omdat ik in Nederland woon.
(Я изучаю нидерландский, потому что живу в Нидерландах)
```
##### **Doordat (из-за того что)**
```
Doordat het regent, blijf ik thuis.
(Из-за того, что идет дождь, я остаюсь дома)
```
##### **Daarom (поэтому)**
```
Het regent, daarom blijf ik thuis.
(Идет дождь, поэтому я остаюсь дома)
```
#### 2⃣ **Противопоставление (Tegenstelling)**
##### **Maar (но)**
```
Ik wil uitgaan, maar het regent.
(Я хочу пойти гулять, но идет дождь)
```
##### **Hoewel (хотя)**
```
Hoewel het regent, ga ik naar buiten.
(Хотя идет дождь, я иду на улицу)
```
##### **Ondanks dat (несмотря на то что)**
```
Ondanks dat hij ziek is, gaat hij werken.
(Несмотря на то, что он болен, он идет работать)
```
#### 3⃣ **Условие (Voorwaarde)**
##### **Als (если)**
```
Als het mooi weer is, gaan we picknicken.
(Если будет хорошая погода, мы пойдем на пикник)
```
##### **Tenzij (если не/кроме как)**
```
Ik kom, tenzij ik ziek ben.
(Я приду, если не буду болен)
```
#### 4⃣ **Время (Tijd)**
##### **Terwijl (пока/в то время как)**
```
Terwijl ik kook, kijk jij televisie.
(Пока я готовлю, ты смотришь телевизор)
```
##### **Nadat (после того как)**
```
Nadat ik gegeten heb, ga ik wandelen.
(После того как поем, пойду гулять)
```
##### **Voordat (прежде чем)**
```
Voordat je vertrekt, bel je me.
(Прежде чем уйдешь, позвони мне)
```

## Порядок слов в сложных предложениях

В голландском различают два базовых паттерна:

- **Hoofdzin (главное)** — глагол на **втором** месте (V2): `Ik blijf thuis omdat het regent`.
- **Bijzin (придаточное)** — после союза (`omdat`, `dat`, `als`, `terwijl`, `hoewel`, `nadat`, `voordat`) **все глаголы уходят в конец**: `..., omdat het vandaag hard regent`.

Если предложение **начинается** с bijzin, сразу после запятой в hoofdzin идёт **глагол + подлежащее** (инверсия): `Omdat het regent, blijf ik thuis`.

## Главное + придаточное предложение

```
[Hoofdzin] + [Bijzin]
Ik denk + dat hij gelijk heeft.
(Я думаю, что он прав)
```

## Придаточное + главное предложение

```
[Bijzin] + [Hoofdzin]
Omdat het regent, + blijf ik thuis.
(Потому что идет дождь, я остаюсь дома)
```

## Важно: В придаточном глагол уходит в конец

Сравни:
- Верно: `Ik denk dat hij morgen **komt**.` (bijzin → глагол в конце)
- Неверно: `Ik denk dat hij komt morgen.`

Если в bijzin несколько глаголов — их **все** в конец, в порядке "модальный/вспомогательный + инфинитив/причастие":
- `..., omdat ik het **heb gezien**.`
- `..., omdat ik niet **kan komen**.`

## Ролевая игра

**Ситуация**: Вы планируете отпуск с другом. Обсудите планы, используя сложные предложения.
**Полезные фразы**:
- Als het weer goed is, kunnen we...
- Omdat we niet veel geld hebben...
- Hoewel ik liever naar... wil...
- Nadat we de tickets hebben geboekt...
- Tenzij je andere plannen hebt...

## Рабочие переговоры в Нидерландах

В Нидерландах важно четко выражать причинно-следственные связи в деловом общении:
- **Omdat** часто используется для объяснения решений
- **Hoewel** показывает, что вы рассмотрели альтернативы
- **Als** используется для постановки условий в переговорах
- **Daarom** помогает сделать логические выводы
**Пример деловой речи**:
```
"Omdat de markt verandert, moeten we onze strategie aanpassen.
Hoewel dit risico's met zich meebrengt, denk ik dat we daarom
sneller moeten handelen. Als we nu beginnen, kunnen we als eerste
op de markt zijn."
```
