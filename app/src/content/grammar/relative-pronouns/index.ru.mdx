Относительные местоимения связывают придаточное предложение с существительным. Выбор зависит от рода антецедента (de/het) и роли в предложении: die — для de-слов, dat — для het-слов, wat — для неопределённых и после alles/iets/niets, waar+предлог — для предметов и мест.

**de-woord → die | het-woord → dat | неопределённое/всё-ничто → wat | предмет с предлогом → waar+предлог (одним словом)**

Придаточное с относительным местоимением имеет порядок слов придаточного: глагол в конце. Относительное местоимение стоит в начале придаточного, далее — подлежащее и остальные элементы. Для людей можно использовать die/dat; для вещей с предлогом вместо 'voor wie/dat' используется слитная форма waarvoor, waarmee, waarover и т.д.

- die — de-слова (de man, de vrouw, de kinderen): De man die daar staat, is mijn oom.
- dat — het-слова (het huis, het kind): Het huis dat ik koop, is klein.
- wat — после alles/iets/niets/het enige: Alles wat hij zegt, is waar.
- wat — когда антецедент — целая фраза: Hij komt niet, wat jammer is.
- waar+предлог — для предметов с предлогом: De stoel waarop ik zit, is nieuw. (= De stoel waar ik op zit / waarop ik zit)

### Для людей — die / wie

<Tip title="Простое правило выбора">
- Сначала смотри на антецедент: de-слово или het-слово?
- de-слово → die (в ед. и мн. числе)
- het-слово → dat (только в ед.ч.!); во мн.ч. het-слова становятся de-словами → die
- Если антецедент — alles/iets/niets/het enige/het mooiste → wat
- Если антецедент — вся фраза → wat
- Если нужен предлог с предметом → waar+предлог (слитно)
- Если нужен предлог с человеком → предлог + wie (раздельно)
</Tip>

<Tip title="Порядок слов в придаточном">
- Придаточное с die/dat/wat/waar+prep — это bijzin.
- Порядок: die/dat/wat + подлежащее + … + глагол в конце.
- De film die ik gisteren gezien heb, was geweldig.
- Het boek dat mijn zus mij heeft gegeven, is fantastisch.
</Tip>

<Tip title="waarop vs waar ... op (разделяемая форма)">
- Разрешены оба варианта: De tafel waarop het staat / De tafel waar het op staat.
- Слитная форма (waarop) более письменная, разделяемая — разговорная.
- В школьной/академической речи обычно используется слитная.
</Tip>
