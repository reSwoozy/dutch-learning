Валентность — сколько и каких дополнений требует глагол: прямое (lijdend voorwerp), косвенное (meewerkend voorwerp), предложное (voorzetselvoorwerp). Для B2 и NT2 II нужно уверенно различать падежные отношения, чтобы ставить правильный порядок слов и правильно использовать пассив.

**[S] + [V] + [meewerkend voorw. — Aan wie?] + [lijdend voorw. — Wat?] + [voorzetselvoorw.]**

Прямое дополнение отвечает на вопрос 'что? кого?' (lijdend voorwerp — 'страдающий объект'), косвенное — на вопрос 'кому? для кого?' (meewerkend voorwerp — 'участвующий'). Глагол может иметь 0, 1, 2 или 3 дополнения. Пассивизировать можно только глаголы с прямым дополнением.

- Проверка: 'Ik geef [mijn vriend] [een cadeau].' → meewerkend (mijn vriend), lijdend (een cadeau).
- Если объект вводится предлогом 'aan/op/voor' — это voorzetselvoorwerp (не прямое).
- В пассиве субъект превращается в 'door + agent': Het cadeau wordt (door mij) gegeven aan mijn vriend.
- В B2-тексте указывайте явно всех участников (кто делает, кому, что).

<Tip title="Как определить, прямое или косвенное">
- Вопрос 'Wat? / Wie?' → lijdend voorwerp (прямое дополнение).
- Вопрос 'Aan wie? / Voor wie?' → meewerkend voorwerp (косвенное дополнение).
- Дополнение с предлогом aan/op/voor/over (фиксированным) → voorzetselvoorwerp — по сути часть глагола.
</Tip>

<Tip title="Порядок объектов в тексте B2">
- Короткое известное перед длинным новым: 'Ik geef 'm (hem) een groot, opvallend cadeau.'
- В формальном тексте с 'aan + mv.voorw.' порядок свободнее: 'We bieden een interessante stage aan studenten met ervaring.'
- Для фокуса на получателе — 'aan ...' в конец.
</Tip>
