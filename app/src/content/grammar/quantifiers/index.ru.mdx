Слова, выражающие количество без точного числа: veel (много), weinig (мало), wat (немного), een paar (пара), enkele (несколько), genoeg (достаточно), alle (все), geen (ни одного). У большинства есть неизменяемая и склоняемая формы, и каждое сочетается с определённым типом существительных (исчисляемые / неисчисляемые).

**veel / weinig — перед неисчисл. и мн.ч. | veel + e перед прил.+ de/het-сущ.: vele mooie boeken | een paar / enkele / sommige — только мн.ч. | wat + неисчисл. | genoeg чаще ПОСЛЕ сущ.: geld genoeg**

Количественные делятся на две группы: (1) для неисчисляемых (veel melk, weinig geld, wat brood, genoeg tijd); (2) для исчисляемых во множественном (veel mensen, enkele boeken, een paar dagen, sommige dagen). 'veel' и 'weinig' работают в обеих группах. Большинство количественных как адъективный определитель получают -e перед de/het-сущ. в определённом контексте, но в стандартной письменной речи 'veel' и 'weinig' чаще остаются без окончания.

- Неопределённое множество: veel mensen / weinig tijd / een paar vrienden.
- Достаточность: genoeg geld / tijd genoeg / voldoende informatie.
- Полнота: alle huizen / allemaal / helemaal.
- Отсутствие: geen boek (ни одной книги / нет книг).

<Tip title="veel vs vele">
- В большинстве случаев 'veel' без окончания: veel mensen, veel boeken.
- 'vele' с -e — формально-литературная форма перед мн.ч. с определённым значением: de vele mogelijkheden, in vele opzichten.
- В учебной речи можно придерживаться 'veel' — это всегда правильно.
</Tip>

<Tip title="wat vs een paar">
- 'wat' — разговорное 'немного'; для исчисляемых во мн.ч. в письме лучше 'een paar' или 'enkele'.
- 'wat mensen' — допустимо в речи, но 'een paar mensen' звучит чище.
- 'wat geld' (немного денег) — полностью нейтрально для неисчисляемых.
</Tip>

<Tip title="geen — количественное отрицание">
- 'geen' заменяет 'een' в отрицании: Ik heb een boek → Ik heb geen boek.
- 'geen' также заменяет отсутствие артикля: Ik drink koffie → Ik drink geen koffie.
- Перед мн.ч.: Ik heb geen kinderen (нет детей).
- Перед прилагательным во мн.ч.: geen grote problemen.
</Tip>
